ECA&D indices
=============

icclim 5.2 comes with convenience functions to call each individual index from `icclim` namespace.
These functions are autogenerated and are stored in `_generated_api.py`.
Additionally, user custom indices can be called with the `icclim.custom_index`.

For example to use this new API with the index `su` you can do:

.. code-block:: python

    import glob
    import icclim

    summer_days = icclim.su(in_files=glob.glob("netcdf_files/tasmax*.nc"))
