What is a climate index?
========================
A climate index is a calculated value that can be used to describe the state and the changes in the climate system. 

Indices calculated by :mod:`icclim`:
------------------------------------

+-----------------+-------------------------------------------------------------------+-----------------------------------------------------------------+
| Short name      | Long name                                                         | Description                                                     |
+=================+===================================================================+=================================================================+
| .. _csdi:       | Cold-Spell Duration Index                                         | Number of days with at least six consecutive days where the     |
| CSDI            |                                                                   | daily minimum temperature is below the 10th percentile.         |
+-----------------+-------------------------------------------------------------------+-----------------------------------------------------------------+
